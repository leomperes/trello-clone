# TRELLO clone

Demo of Trello using Rails and Vue.js.


## Quick Start

1.  Setup your project gems/libraries environment installing [RVM](http://rvm.io).

    ```shell
    $ rvm install 2.5.3
    $ rvm use 2.5.3@trello-clone --create
    ```

2.  Install libraries.

    ```shell
    $ bundle install
    $ yarn install
    ```

3.  Prepare the database:

    ```shell
    $ rails db:setup
    ```

5.  **Run the server**

    ```shell
    $ rails s
    ```
